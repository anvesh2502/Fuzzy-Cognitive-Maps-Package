§ hebbian_learning()

def hebbian_learning(fcm, restraints, stabilizers, transferFunct, neu, k = maxsize)

Parameters :
  • fcm - An object of the FCM
  • restraints - dictionary that contains the range of the desired output of concepts
  • stabilizers - concepts to stabilize on and their stabilization threshold
  • transferFunct - a function to keep values in desired range
  • neu - a float in range [0,1] that is the learning coefficient
  • k - default is max system size, how many steps the system should use to learn

Returns : The new edge matrix for the FCM
Return Type : A list of lists
Exceptions which can be raised by this method : None 
This method will take an FCM and using the Nonlinear Hebbian Learning method to get optimal edge values for an FCM starting from an initial state.

§ updateConcepts()

def updateConcepts(edgeMatrix, nodeValues, transferFunction, stableList)

Parameters :
  • edgeMatrix - A new matrix of edges
  • nodeValues - a list in node order that matches the edge matrix
  • transferFunction - a function to keep values in range[-1,1]
  • stableList - list of the concepts with no incoming edges

Returns : the updated values of the nodes after one time step
Return Type : A list of lists
Exceptions which can be raised by this method : None 
This method will convert the list into a numpy array and multiply it with the edge list to get the changes to each node.
The changes will be applied to all nodes to the transfer function. If a node has no incoming edges it is declared stable and returned to its old value.
