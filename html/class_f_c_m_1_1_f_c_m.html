Detailed Description

Fuzzy Cognitive Maps are digraphs where the results of the events and the effect of these events on other events are depicted
 along with the strength of impact.This class is the data structure for the FCM where the creation and the manipulation of the graph happens.

def FCM.FCM.add_edge(concept1,concept2,weight) - Adds an edge between two concepts
Parameters :
  • concept1(string) - Name of the first concept
  • concept1(string) - Name of the second concept
  • weight (int)- An integer in the bounds [-1,1]
Returns : None
Return Type : None
Exceptions which can be raised by this method : InvalidWeightError 
This method will add an edge between two concepts and will also accept a weight of the edge.The
important notable points while using this function are :
-- If any of the two concepts do not exist,this method will create them before adding an edge
-- If the weight parameter is out of bounds,InvalidWeightError will be raised.
The proper usage of this function is given in the code snippet below


try :
  fcm.add_edge("concept1","concept2",0.8)
except InvalidWeightError :
  print('You are seeing me because of the invalid weight value') # Here is where you reach when the weight value is out of bounds

§ set_value()

def set_value(concept,value) - sets a value to a concept in the fcm
Parameters :
  • concept(string) - Name of the concept
  • concept(string) - Name of the second concept
  • value (int)- An integer in the bounds [-1,1] or a function with no parameters and returns a value between [-1,1]
Returns : The value of the concept
Return Type : int
Exceptions which can be raised by this method : ConceptExistError  InvalidConceptValueError
This method is used to set the value of a concept .The value can be set in two ways.
-- Using a variable
-- Using a function 
This method will raise a ConceptExistError Exception if the concept does not exist.
It will raise the InvalidConceptValueError Exception if the value is out of bounds.

The proper usage of this method will be shown in the code snippet below.


# Setting value with a variable
val=0.9
try :
   fcm.set_value('concept1',val)
except ConceptExistError :
   print 'You will see this code if the concept does not exist'

# Setting value with a function

def f() : return 0.5
try :
   fcm.set_value('concept1',f)
except ConceptExistError :
   print 'You will see this code if the concept does not exist'
