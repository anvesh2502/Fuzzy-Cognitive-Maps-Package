Fuzzy Cognitive Maps
Fuzzy Cognitive Maps Documentation

  A Python package for Fuzzy Cognitive Maps

    Fuzzy Cognitive Maps are graphs which are used for finding the impact between relations and can be used to measure the 'strength of impact' between relations. This package is written in Python and provides     various operations such as the creating,drawing and performing learning using algorithms like Hebbian Learning,Particle Swarm Optimization.

    Following is a walk-through of how to use this package step by step.It explains the various steps in using the FCM using coding example snippets.

# FCM Creation
# We are going to create an FCM object in this section by calling the constructor of the FCM class which is in the FCM.py file
from FCM import FCM
fcm=FCM()  # As long as there is enough memory available,this line of code will execute without any exception.
print type(fcm)  



# Adding concepts to FCM
# In this section,we are going to add concepts to an existing valid FCM object

fcm.add_concept("concept1")
fcm.add_concept("concept2") # If you add a single concept more than one time,it will be overridden



# Adding an edge between two concepts in an FCM

# In this section,we are going to add edges between two concepts in different scenarios

# Adding edges between concepts which exist
fcm.add_concept("concept1")
fcm.add_concept("concept2")
weight=0.4 # The weight for the edge 
try :
  fcm.add_edge("concept1","concept2",weight)
except InvalidWeightError :
  print 'You are seeing this code because of the invalid weight'

# When we add edges between concepts that do not exist,this method will create those edges first before adding the edge
